To use keyword search, enter your keywords separated by a single space (e.g., apple banana cherry).

To use wildcard search, start with / followed by your wildcard expression.

The following wildcard characters are supported:
*: Matches zero or more characters. Example: a* matches "apple," "apricot," etc.

?: Matches exactly one character. Example: b?nana matches "banana," "b3nana," etc.

[seq]: Matches any one character within the brackets. Example: c[aou]t matches "cat," "cot," or "cut."

[!seq]: Matches any one character not in the brackets. Example: d[!aeiou]g matches any "dxg" where x is not 'a','e','i','o','u'

%: Matches zero or more characters (used in some systems). Example: a% matches "apple," "apricot," etc.

_: Matches exactly one character (used in some systems). Example: b_nana matches "banana," "b1nana," etc.
